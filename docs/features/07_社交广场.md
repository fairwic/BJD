# 07 社交广场

## 概述
社交内容模块，用户发布动态、互动、关注其他用户。

**优先级**：P1 (Growth)

**包含页面**：
- `CreatePost.jsx` - 发布动态
- `PostDetail.jsx` - 动态详情
- `PublicUserProfile.jsx` - 用户主页
- `MyFollowing.jsx` - 我的关注
- `MyLikes.jsx` - 我的喜欢
- `MyAlbum.jsx` - 我的相册

---

## 1. 发布动态 (`CreatePost.jsx`)

### 1.1 元素交互

| 元素 | 操作 | 响应 |
|------|------|------|
| 图片 | 点击"+" | 相册多选1-9张 -> 九宫格展示 -> 长按拖拽排序 |
| 文本 | 输入 | 最多500字，支持Emoji |
| 标签 | 输入# | 输入#话题#格式 |
| 关联商品 | 点击 | 搜索商品 -> 选中后底部显示卡片 |
| 发布 | 点击 | 乐观更新(立即插入广场) -> 后台上传 |

---

## 2. 动态详情 (`PostDetail.jsx`)

### 2.1 互动功能

| 功能 | 操作 | 响应 |
|------|------|------|
| 点赞 | 点击❤️ | 变红+数字+1；防抖0.5s提交 |
| 评论 | 输入发送 | 立即追加到列表 |
| 分享 | 点击 | 生成卡片/系统分享 |

---

## 3. 用户主页 (`PublicUserProfile.jsx`)

| 区块 | 内容 |
|------|------|
| 顶部 | 头像+昵称+签名+关注/粉丝数 |
| Tab | 动态 / 喜欢 |
| 操作 | 关注按钮、私信按钮 |

---

## 4. 我的关注 (`MyFollowing.jsx`)

Tab：团长/店铺 | 用户

列表：头像+名称+粉丝数

---

## 5. 我的喜欢 (`MyLikes.jsx`)

Tab：商品 | 动态

列表：对应收藏内容

---

## 6. 我的相册 (`MyAlbum.jsx`)

网格展示图片/视频，点击预览/播放
