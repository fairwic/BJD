## 2. 肤色比对室 (`SkinMatchStudio.jsx`)

### 2.1 功能定位
可视化比对不同品牌的肤色，解决"配身"难题。

### 2.2 页面结构
```
┌─────────────────────────────┐
│ [<]       肤色比对室         │
├─────────────────────────────┤
│  [左侧滚轮]   |   [右侧滚轮] │
│  Volks-普肌  |   Luts-普肌  │
├─────────────────────────────┤
│  ┌─────────────────────┐    │
│  │  半圆拼合效果图      │    │
│  │  #fdf4f0 | #fffbf0  │    │
│  └─────────────────────┘    │
├─────────────────────────────┤
│  评分仪表盘: 85分            │
│  评价: 色差较小              │
│  建议: Luts偏粉，请参考实拍  │
└─────────────────────────────┘
```

### 2.3 用户操作

| 步骤 | 操作 | 响应 |
|------|------|------|
| 1 | 左滚轮选择品牌A+肤色 | 左侧半圆更新颜色 |
| 2 | 右滚轮选择品牌B+肤色 | 右侧半圆更新颜色 |
| 3 | 实时计算 | 中间区域渲染拼合效果 -> 仪表盘更新评分 |

### 2.4 评分规则

| Delta E值 | 评价 |
|-----------|------|
| < 2 | 完美匹配 |
| 2-5 | 色差较小 |
| > 5 | 色差明显，不推荐 |
