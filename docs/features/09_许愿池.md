# 09 许愿池

## 概述
许愿池是用户发布心愿(希望再贩的绝版商品)、投票助燃、团长响应的C2B反向定制模块。

**优先级**：P2 (Experience)

**包含页面**：
- `WishPool.jsx` - 许愿池

---

## 1. 许愿池 (`WishPool.jsx`)

### 1.1 页面结构
```
┌─────────────────────────────┐
│ [<]         许愿池           │
├─────────────────────────────┤
│  [我的心愿] [热门榜单]        │
├─────────────────────────────┤
│  ┌─────────────────────┐    │
│  │ Volks 10周年复刻     │    │
│  │ 🔥 9999+ 票          │    │
│  │ [投票] [助燃] [分享] │    │
│  │ 状态: 洽谈中         │    │
│  └─────────────────────┘    │
│  ...                        │
├─────────────────────────────┤
│  [+] 发布心愿 (悬浮按钮)     │
└─────────────────────────────┘
```

### 1.2 核心功能

| 功能 | 操作 | 响应 |
|------|------|------|
| 发布心愿 | 点击"+"按钮 | 弹窗输入商品名称+描述 -> 发布成功 |
| 投票 | 点击❤️ | 票数+1；每日限1票 |
| 助燃(Boost) | 点击🚀 | 弹窗消耗金币 -> 热度+N(倍增效果) |
| 团长响应 | 团长点击"我有资源" | 心愿状态变"洽谈中" -> 通知所有投票用户 |
| 分享 | 点击分享 | 生成拉票卡片 + 专属口令 |

### 1.3 心愿状态流转
```
投票中 -> 洽谈中 -> 开团成功
```

### 1.4 业务规则

| 规则项 | 说明 |
|--------|------|
| 每日投票限制 | 每用户每日1票(免费) |
| 助燃效果 | 消耗10金币 = 热度+20 / 票数+50 |
| 团长响应 | 点击后锁定24小时，需在24h内联系官方 |
