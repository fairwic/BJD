# 08 娃娃档案

## 概述
娃娃档案是用户管理自己拥有的BJD娃娃的数字化模块。

**优先级**：P2 (Experience)

**包含页面**：
- `MyDolls.jsx` - 我的娃娃列表
- `DollProfile.jsx` - 娃娃档案详情

---

## 1. 我的娃娃列表 (`MyDolls.jsx`)

### 1.1 页面结构
```
┌─────────────────────────────┐
│ [<]       我的娃娃     [+]   │
├─────────────────────────────┤
│  ┌─────┐ ┌─────┐ ┌─────┐   │
│  │小鹿 │ │小兔 │ │ +   │   │
│  │3分  │ │4分  │ │添加 │   │
│  │182天│ │54天 │ │     │   │
│  └─────┘ └─────┘ └─────┘   │
└─────────────────────────────┘
```

### 1.2 用户操作

| 操作 | 响应 |
|------|------|
| 点击卡片 | 跳转DollProfile(id) |
| 点击"+" | 跳转添加娃娃页面 |
| 长按卡片 | 显示快捷菜单(编辑/删除/分享) |

## 2. 娃娃档案 (`DollProfile.jsx`)

### 2.1 页面结构
```
┌─────────────────────────────┐
│ [<]     [编辑] [分享]        │
├─────────────────────────────┤
│  [大照片]                    │
│  小鹿                        │
│  龙魂 · 3分 · 白肌           │
│  生日: 2024-01-01            │
│  已养: 182天 💖               │
├─────────────────────────────┤
│  📷 照片 12  👗 衣服 8  🔧 保养 3│
├─────────────────────────────┤
│  关联订单: O12345 [查看]      │
│  电子合同: CT-O12345 [查看]   │
├─────────────────────────────┤
│  保养记录列表...             │
└─────────────────────────────┘
```

### 2.2 功能交互

| 功能 | 操作 | 响应 |
|------|------|------|
| 分享电子身份证 | 点击右上角分享 | 生成精美证书图片(拟物化设计) -> 保存/分享 |
| 查看关联订单 | 点击 | 跳转OrderDetail |
| 添加保养记录 | 点击"+" | 弹窗输入日期+类型+备注 |

### 2.3 电子身份证
证书式设计：照片+姓名+品牌+生日+监护人(当前用户)+ID编号
