# 10 游戏化成就

## 概述
游戏化成就系统，激励用户参与平台活动，解锁徽章和奖励。

**优先级**：P2 (Experience)

**包含页面**：
- `AchievementCenter.jsx` - 成就中心

---

## 1. 成就中心 (`AchievementCenter.jsx`)

### 1.1 页面结构
```
┌─────────────────────────────┐
│ [<]        成就中心          │
├─────────────────────────────┤
│  总进度: 2/8 已解锁 [25%]    │
│  [████░░░░░░]               │
├─────────────────────────────┤
│  [全部] [新手] [购物] [社交] │
├─────────────────────────────┤
│  ┌─────────────────────┐    │
│  │ 🎉 入坑徽章 [已解锁] │    │
│  │ 完成首次参团         │    │
│  │ 奖励: 10元优惠券     │    │
│  └─────────────────────┘    │
│  ┌─────────────────────┐    │
│  │ 💎 壕无人性 [未解锁] │    │
│  │ 累计消费10,000元     │    │
│  │ 进度: 3,500/10,000   │    │
│  └─────────────────────┘    │
└─────────────────────────────┘
```

### 1.2 成就分类

| 类别 | 成就示例 |
|------|----------|
| 新手 | 入坑徽章、实名玩家 |
| 购物 | 剁手王(5次参团)、壕无人性(消费1万) |
| 社交 | 晒图达人(发布10条)、人气王(100赞) |

### 1.3 稀有度系统

| 等级 | 名称 | 视觉效果 |
|------|------|----------|
| 1 | 普通 | 灰色渐变 |
| 2 | 稀有 | 蓝色渐变 |
| 3 | 史诗 | 紫色渐变+动态光效 |

### 1.4 解锁动画
达成成就 -> 全屏Lottie动画(烟花/钻石雨) -> Toast"恭喜获得XXX" -> 徽章从黑白变彩色

### 1.5 奖励类型
- 虚拟徽章(个人主页展示)
- 优惠券(10-200元)
- 信用分加成(+5~30)
